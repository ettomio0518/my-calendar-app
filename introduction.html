舞いカレンダー 紹介ページ

マイカレンダーと読みます。従来の「音声カレンダー」の機能に加えて、簡易的にGoogleカレンダーに進めるようにしました。これによりハイブリッド型カレンダーアプリに生まれ変わりました。
さらに最強ツール「今日のカード」を搭載しました。

音声で予定をお知らせ！
シンプル操作で使える多機能カレンダーアプリです。

📱 簡易版の主な機能

  • カレンダー表示（祝日色分け、2020〜2040年対応）
  • 日付をタップで予定を音声で確認
  • 祝日をタップで祝日名を音声で確認
  • 10件までの予定登録

🌟 プレミアム版の拡張機能

🆕 「今日のカード」機能 強化！

予定・天気・メモ・日記を「今日」1ページに集約。
その日に必要な情報を、迷わず・静かに確認できる最強ツールです。

  • ⏰ 時間ブロックをタップ
    予定内容を アラート＋音声 でお知らせ。
    視覚と聴覚の両方で確認でき、スマホでも直感的に操作できます。
  • 📣 直近の予定を自動判定
    現在時刻から一番近い予定を検知し、アラート＋音声通知。
    まるで秘書のように「次」を教えてくれます。
  • 📅 時間割グラフ表示
    一日の流れをひと目で把握。
    予定の重なりも回避済みで、安心して使えます。
  • 🌤️ 暦・天候情報をまとめて表示
    天気／六曜／星座／和風月名まで統合。
    実用性と日本的な楽しさを両立しました。
  • ✍️ 日記＆走り書きメモを連携
    ・走り書き：ひらめき、買い物メモなど自由に記録
    ・7日以上前の走り書きは削除可能
    ・日記機能も、より使いやすく改善されています

📌 「今日」に必要な情報を、1画面で。
考えなくても、開けば分かる。
舞いカレンダーが目指す「静かな便利さ」を体現した機能です。

🎙️ 音声入力がさらに進化しました！

予定入力や高機能検索の音声入力にて、最大3つまでの変換候補を表示するようになりました。

これにより、認識精度が向上し、聞き間違いや入力ミスにも柔軟に対応できます。より快適で、誰でも使いやすい音声操作を実現しています。

💻 Web版をご利用の方へ（PCブラウザ）

パソコンのブラウザからWeb版をご利用の際は、音声入力ではなく手動入力を推奨しています。

これは、音声認識に使用している Web Speech API が、PC環境ではブラウザやOSの制限により安定して動作しない場合があるためです。

（不安定になる理由）
音声認識処理は、マイクの音声を**外部の高性能なサーバー**に送信し、その結果をリアルタイムで受け取る仕組みです。PCのブラウザ環境では、他のアプリケーションやネットワークのわずかな途切れの影響を受けやすく、サーバーとの接続が不安定になり、「音声認識エラー」が頻繁に発生することが確認されています。

そのため、確実な操作と快適な入力のために、手動入力をご利用ください。

  • 予定登録は無制限（音声入力を含む全ての入力に対応）

  • 📱 iPhoneでWeb版の音声入力機能をご利用の方へ

    音声入力機能は、AppleのSafariブラウザでSafari 14.1以降（iOS 14.5以降に相当）でのみ動作します。ご利用前に、Safariのバージョンをご確認ください。
    🚨 重要なご注意
    • 必ずSafariブラウザでWebサイトを直接開くか、ブックマークからアクセスしてください。
    • 「ホーム画面に追加」した状態（PWA）では、音声入力は動作しません（Appleの仕様による制限です）。
    • 音声入力が使えない場合は、手動入力をご利用ください。
  • 予定のある日はカレンダーに色付き表示
  • 時刻の音声読み上げ付きアナログ時計
  • 割引・税込み対応のお買い物電卓
  • 6分類対応のメモ機能と簡易日記（一覧表示）
  • キーワード検索・年齢計算・誕生日メッセージ
  • ブックマーク・通常電卓など便利ツールも充実
  • マスコット「カレミン」をクリックすると主な機能に直接アクセス可能
  • データの保存と復元機能を搭載
  • 世界の時刻表示を搭載
  • 現在地の天気予報を搭載　ヘッダーの「舞い ♪ カレンダー」クリックで表示
  • 今日のカードを搭載　ヘッダー下の今日の赤文字の日付(例 2025/12/11)をクリックで表示
  • カレンダーで、例 「9時から13時までテスト」と入力すると、それが今日ならば自動的に「今日のカード」の「今日の時間割」に反映されます。
  • 「今日の時間割」で表示される時間ブロックをタップで予定内容をアラートと音声出力します。
  • 「直近の予定出力」で現時刻に一番近い予定内容をアラートと音声で出力します。
  • 「今日の走り書き」で今日の買い物メモなどをアラートと音声で確認できます。

🔊 注意事項とヒント

  • 端末の音声読み上げ（TTS）機能が必要です
  • 日本語読み上げに対応
  • 不要になった予定は削除を推奨（容量節約）
  • **削除したくない過去の行事などは残すことにより、日付に色は残ります**
  • 天気予報表示のために端末の位置情報を利用します。位置情報は端末内でのみ使用され、外部に送信・共有されることはありません。
  • 個人情報の入力はご遠慮ください
